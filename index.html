How tournaments work

From sign-up to final match — here's what happens when you join a tournament.

✍️

Registration & Check-In

Players register with their Riot ID. Raven verifies the account via the Riot API automatically. A check-in system confirms attendance before the tournament locks in.

⚖️

Skill-Based Team Balancing

Teams are balanced using ranked data from the Riot API — no lopsided games.

🗺️

Visual Brackets

Auto-generated bracket images update in real time as matches are reported. Supports single and double elimination.

🎮

Role Assignment

Players set preferred roles (Top, Jungle, Mid, ADC, Support). Raven assigns roles across teams to maximize satisfaction.

👥

Substitute Pool

No-shows at check-in are moved to a sub pool. If a player drops mid-tournament, subs are offered the spot instantly.

📋

Live Dashboard

A pinned dashboard shows real-time state — registered players, team assignments, match results, and current standings.
